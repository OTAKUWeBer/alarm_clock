# Async Alarm Clock

Async Alarm Clock is a Python command-line application for setting and managing alarms asynchronously.

## Features

- **Set Alarms**: Set alarms with a name and duration (in minutes and seconds).
- **Manage Alarms**: View active alarms and their remaining time.
- **Remove Alarms**: Remove alarms by name.
- **Alarm Sound**: Plays a sound when an alarm goes off.

## Getting Started

### Prerequisites

- Python 3.7 or higher

### Installation

Clone the repository:

```
git clone https://github.com/OTAKUWeBer/alarm_clock.git
cd alarm_clock
```

Install dependencies:

```
pip install pygame aioconsole
```

### Usage

Run the application:

```
python alarm_clock.py
```

### Commands

1. **Add alarm**: Set a new alarm by entering a name and duration.
2. **Check alarm lists**: Display active alarms and their remaining time.
3. **Remove alarm**: Remove an alarm by its name.
4. **Exit**: Quit the application.
